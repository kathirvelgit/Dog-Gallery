# 🐶 Dog Gallery

A simple and responsive **Dog Gallery** built using **HTML5, CSS3, and JavaScript**. Click the **Generate** button to instantly load a new adorable dog image. The project demonstrates DOM manipulation, event handling, and dynamic image generation while preventing browser caching.

---

## 🚀 Features

- 🐕 Generate random dog images with a single click
- 🎯 Responsive and clean user interface
- 🖼️ Dynamic image loading using JavaScript
- ⚡ Prevent browser image caching using timestamps
- 🎨 Modern button hover animation
- 📱 Mobile-friendly responsive design

---

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)
- 🐶 PlaceDog Image Service
- 🅱️ Bootstrap 5
